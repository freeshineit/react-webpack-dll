## install

```sh
git clone git@github.com:freeshineit/react-webpack-dll.git

cd react-webpack-dll

npm install

#或者

yarn install 

# 开发环境
npm run dev

# 单独打包依赖包文件， 一次打包全局使用， 当包更新时需重新打包
npm run dll 

# 生产环境
npm run build

```
